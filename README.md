# Amalia Elena Mihoc — Interactive Resume

A clean, minimal, and dynamic personal CV website built with HTML, CSS, and JavaScript.

## Features

- **Bilingual** — toggle between English and French with a single click
- **Animated entrance** — smooth fade and slide-in on page load
- **Scroll-triggered reveals** — sections animate in as you read down
- **Hover effects** — subtle interactions on job entries and skills
- **Animated skill bars** — proficiency levels animate into view on scroll
- **Print-ready** — clean output when printed directly from the browser
- **Fully self-contained** — single HTML file, no dependencies or build steps

## Live Demo

[View CV →](https://amaliatomaxyz.github.io/my-interactive-resume)

## Tech Stack

- HTML5
- CSS3 (custom animations, CSS variables, Grid layout)
- Vanilla JavaScript (IntersectionObserver for scroll reveals)
- Google Fonts — Barlow & Barlow Condensed

## Usage

No setup required. Just open `index.html` in any modern browser.
